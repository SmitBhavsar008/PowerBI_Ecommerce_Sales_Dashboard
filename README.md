# PowerBI_Ecommerce_Sales_Dashboard

## 📊 Project Learnings (Power BI Focused)

* Developed an interactive Power BI dashboard to monitor and analyze online sales performance.
* Performed data modeling by creating relationships between multiple tables using the Power BI data model.
* Applied Power Query (ETL process) for data cleaning, transformation, and preparation.
* Created DAX measures and calculated columns to derive meaningful business insights.
* Implemented slicers, filters, and drill-through/drill-down features for dynamic and user-driven analysis.
* Designed visually appealing reports using various Power BI visualizations such as bar charts, line charts, pie charts, maps, and KPI cards.
* Enabled interactive reporting with user-friendly navigation and customized views.
* Optimized report performance using efficient data modeling and DAX techniques.
